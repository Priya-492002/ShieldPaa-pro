# ShieldPass Pro 🛡️

A modern Password Management Suite built with Python and CustomTkinter.
## 📂 Files in this Project
* **Shieldpass_pro_final.py**: Final version with generator and strength tester.
* **password_suits_v1.py**: Full feature version with slider.
* **strength_tester_basic.py**: Original strength tester prototype.
## 🛠️ How to Use
1. Install the requirements: `pip install -r requirements.txt`
2. Run the main application: `python pro3.py`
